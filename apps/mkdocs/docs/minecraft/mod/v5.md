## Gregtech の Config ファイルを追加

1. mod の入れ終わったら、一度起動しよう。
2. 起動ができたら、一度マイクラを停止して
3. このファイルをクリックしてダウンロードして、解凍しよう。
4. ダウンロードした config ファイルを上書き保存しよう。`~\config`
